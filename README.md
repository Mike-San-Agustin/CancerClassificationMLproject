# Cancer Classification Project
R-Language, Data manipulation, Feature Engineering, PCA, Logistic Regression, Random Forest, XGBoost, Cross-Validation

**For Recruiters**: Download the pre-rendered cancerclassification.html file to see code and outputs. It can run on browsers like Google Chrome.

This repository contains a comprehensive data science project focused on the malignancy-benign diagnosis of breast cancer imaging data. This analysis utilizes several R packages including: tidyverse, tidymodels and corrplot. The purpose of this project is to create predictive models based on the Breast Cancer Wisconsin Diagnostic dataset. Overall, the logistic regression model outperformed other random forest, XGBoost and KNN, achieving a 99.7% diagnosis accuracy and a 98% sensitivity rate.

The repository contents are as follows:

1. **Data Files:**
   • wdbc.data: The raw dataset containing features computed from a digitized image of a fine needle aspirate (FNA) of a breast mass.
   • wdbc.names: Documentation and names of variables in the dataset
2. **Analysis and Code:**
   •cancer_class_data.R: R script for data cleaning and preprocessing
   •cancer_class_EDA.Rmd: R markdown focused on exploratory data analysis.
   •cancerclassification.Rmd: Main R markdown file containing the full modeling pipeline.
   •CancerClassification.RProj: Rstudio project file for environment management.
3. **Outputs:**
   •cancerclassification.html: Rendered report of the analysis.
